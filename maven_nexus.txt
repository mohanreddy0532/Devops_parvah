
What is snapshot?
Final version of the development team

Snapshot vs Version?
Maven can automatically download the latest snapshot while building..
In case of Version Maven doesn't fetch automatically.

What is GAV?
GroupID Id of the project group e.g com.company.bank 
ArtifactID Id of the project e.g consumer-banking, along with groupId defines the artifact's location in Repository
VersionId version of the project e.g:com.project.bank:consumer-banking:1.1
